# sprintgroup

First time use only (new machine):
Note: Make sure in the Git Bash CMD, that you are in the folder you wish to have the repository
           May need to go back cd ../ or forward cd <folder>
1)	git clone https://github.com/slame107/sprintgroup.git
*may prompt you to log into github*
2)	cd sprintgroup
3)	git config user.name “Name”
4)	git config user.email username@uwsp.edu 

Start of work Session:
?	cd sprintgroup (if not already in ~/sprintgroup (master)
?	git pull
*enter in github credentials if prompted*

End of work session:
?	git add sprintgroup/
?	git commit -a -m “Insert message details here”
?	git push
*enter credentials if prompted*
